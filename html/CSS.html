Centering Recipes

Horizontal Centering Fluid Blocks

Horizontal centering of blocks of unkown width

#horizontal-center{
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
#vertical-center{
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
/* Default */
a {
  color: blue;
}
/* Unvisited links */
a:link {
  color: blue;
}
/* Visited links */
a:visited {
  color: purple;
}
/* Hover state */
a:hover {
  color: red; 
}
/* Focused state */
a:focus {
  color: orange;
}
/* Activated state */
a:active {
  color: green;
}

example

This is centered


  • The left property relates to the size of parent
  • The transform property relates to the child size
  • thus it will be centered

Horizontal centering of blocks of kown width

#horizontal-center{
  width: 550px;
  margin: 0 auto;
}
#vertical-center{
  position: absolute;
  top: 50%;
  height: 400px;
  margin-top:-200px;
}
/* vertically align text */
#vertical-center{
  height: 40px;
  line-height:40px;
}

example

This is centered


  • auto makes the browser calculate unkown portion

Background Repeat

background-repeat property allows us to specify if background image will repeat horizontally repeat-x or vertically repeat-y, or use no-repeat to prevent this

body {
  background-image: url(image.jpg);
  background-repeat: repeat-x;
}
